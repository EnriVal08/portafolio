API Reference

Esta sección describe los endpoints principales y ejemplos de uso.

Cada endpoint tiene su método HTTP y parámetros listados.

Los códigos de respuesta comunes son 200, 400 y 500.

  • GET /items — devuelve lista de ítems
  • POST /items — crea un nuevo ítem
GET /api/v1/items?page=1&limit=20
POST /api/v1/items
Content-Type: application/json

{ "name": "Mi ítem" }
